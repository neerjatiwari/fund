Welcome to the Triple Crown for Heart donation site!

The Triple Crown for Heart is an annual cycling ride up the three North Shore mountains in Vancouver, British Columbia.

The ride is also a fundraising event for two worthwhile charities, the BC Children’s Hospital Foundation and the Children’s Heart Network. We encourage our riders to take the time to raise money for these great causes.

This fundraising website was built in Python/Django specifically for the Triple Crown for Heart by Dominik Szopa. The complete source code is available on GitHub under the MIT License. If you are interested in using this platform for your charity (for free), please contact techdomi@gmail.com.
